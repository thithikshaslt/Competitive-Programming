## [Missing Number](https://leetcode.com/problems/missing-number/)

**Level** : Easy

---

### **Approach**

find sum upto n using the formula n*(n+1)/2 and then find the sum of all elts in the list, return the difference 

Time Complexity : O(n) (one for loop)
